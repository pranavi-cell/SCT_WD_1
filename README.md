StreamFlix

StreamFlix is a Netflix-inspired streaming website built using HTML, CSS, and JavaScript. The project features a modern user interface, responsive design, and an interactive navigation menu.

Features
Fixed navigation bar
Navigation menu hover effects
Responsive design for desktop, tablet, and mobile devices
Hero banner section
Trending and Popular content sections
Interactive movie cards with hover animations
Technologies Used
HTML5
CSS3
JavaScript
Font Awesome
Project Structure
streamflix/
│
├── index.html
├── style.css
├── script.js
└── README.md
How to Run
Download or clone the repository.
Open the project folder.
Open index.html in any web browser.

No installation or additional setup is required.

Building responsive web pages
Creating interactive navigation menus
